EVSU-OC-Aliblog: Eastern Visayas State University - Ormoc Campus Automated Library Logbook

The Eastern Visayas State University - Ormoc Campus Automated Library Logbook is a group project developed for our Object-Oriented Programming course. This project modernizes the traditional library logbook by implementing QR code automation, making the logging process faster and more efficient. My responsibility included creating documentation and project reports, ensuring all project requirements were met and progress was well-organized.

Features:

  • QR Code Scanning: Quickly log in/out students and staff using QR codes.
  • Automated Record Keeping: All entries are saved digitally for easy tracking.
  • User Management: Add, edit, or remove library members efficiently.
  • Real-time Updates: Logs are updated instantly, reducing errors and manual work.
  • Reporting: Generate reports of library usage for administration review.

Tools/Languages: PYTHON, TKINTER, SQLITE

My Role: Responsible for documentation and project reports, ensuring that all progress, features, and procedures were well-organized and clearly presented.
